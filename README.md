## 🛠️ Setup Steps for Your Profile

<details>
<summary><b>Click to expand — How to enable the Snake contribution animation</b></summary>

<br/>

The contribution snake is generated via a **GitHub Action**. To set it up:

1. In your `Aayushinba17` repo, go to **Actions** → **New workflow** → **set up a workflow yourself**
2. Name the file `.github/workflows/snake.yml`
3. Paste this content:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 0 * * *"    # runs daily at midnight UTC
  workflow_dispatch:         # allows manual trigger

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: Aayushinba17
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

4. **Commit** the file, then click **Actions** → **Generate Snake** → **Run workflow**
5. After it runs, the snake SVG will be at:
   `https://raw.githubusercontent.com/Aayushinba17/Aayushinba17/output/github-snake-dark.svg`

</details>
